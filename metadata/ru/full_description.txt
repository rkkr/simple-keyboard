Возможности:
* Маленький размер (<1 МБ)
* Регулируемая высота клавиатуры для большего количества экрана
* Цифровой ряд
* Проведите по пробелу чтобы перемещать курсор
* Удалить смахиванием
* Настраиваемая цветовая темы
* Минимальные разрешения (только Вибратор)
* Без рекламы

Функции которых нет и, вероятно, никогда не будет:
* Эмодзи
* GIFки
* Проверка орфографии
* Набор текста без отрыва пальца (Swipe typing)

Чтобы включить эту клавиатуру:
* Откройте приложение Simple Keyboard
* Включите эту Simple Keyboard (будет показано системное предупреждение по умолчанию об отслеживании)
* Переключитесь на Simple Keyboard из текущего метода ввода (отличается между клавиатурами, обычно долгое нажатие на пробел)
* Чтобы настроить Simple Keyboard используйте долгое нажатие на "," или откройте системные настройки, Языки и ввод, Simple Keyboard.
